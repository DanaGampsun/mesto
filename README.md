# Проект работа №4: Место

Сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### [Проектная работа](https://danagampsun.github.io/mesto/)

### Используемые технологии

* HTML
* CSS
* BEM
* Grid Layout
* Flex-box
* Media queries
* Adaptive layout
* JS
